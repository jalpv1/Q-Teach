Завдання:
Розширити проект який бачили на 1 лекції.
Кожна команда має добавити нові можливості (Історія голосування, проходження тестів, .....)
Функціонал в команд можуть повторюватися!
Завдання для практик
1) Ознайомитися з Spring, Spring Shell, DI.
Створити трьох рівневу архітектуру: Simple Data Source (in memory) <- Domain (Бізнес логіка) <- Presentation.
Продумати які нові можливості ви б хотіли реалізувати та реалізувати їх.
2) Заміняємо Prsentation Layer розробляємо REST API (Spring Shell -> Spring MVC).
Отримуємо проект (voter який бачили на лекції).
3) Заміняємо Data Source Layer підключаємо Spring Data (ORM). Добавляємо базу даних.
4) Створюємо WEB Client для REST API (Angular 2+).
